• Angular Bootcamp

    After this course participants will be able to read and understand an angular codebase, and figure out right components to implement a funcitonality.
    This course will demonstrate how to use following angular components :
    • modules
    • templates
    • directives
    • controllers and scope
    • filters
    • services
    • models
    • ui.router
    • $http
    2 Days

  • Test Driven Development With Angular

    This course will give participants hands-on experience with test driven development with angular. Also participants will learn best tools and practices for angular development, like task runners, dependency management, unit testing
    This course will demonstrate how to use following angular components :
    • npm
    • gulp
    • bower
    • jasmine
    • loading modules
    • testing directives
    • testing controllers
    • testing filters
    • testing services
    • testing models
    • testing routes
    3 Days

  • Behavior Driven Development With Angular

    This course will demonstrate how to use business requirements to drive your development. Participants will be given hands-on experience with writting end to end tests for angular client (including HTML and user actions), stubbing remote server, and using gherkins.
    This course will use following :
    • gherkin
    • cucumber
    • selenium
    • webdriver
    • expressjs
    2 Days

  • Interaction Design With Angular

    This course will give hands-on on experience on creating custom UI components and interaction using Angular with SCSS and CSS3. This will also talk about how browser renders a page and how to create smooth animations that match to the performance of a native app. Also it will give tips/tricks and hacks for ensuring better user experience.
    This course will use following :
    • Browser Rendering Cycles
    • CSS3 transformations and animations
    • SCSS
    • User Experience Design
    • Creating custom UI components
    2 Days
