Carousel with JQuery

Introduction

In this lesson we will implement a carousel with jQuery. The same can also be don with pure javascript, but we use jQuery to ensure cross browser compatibility. You need moderate skills with js, css and html5.

Design

We will implement the carousel as an array of slides. Slides are enclosed in a container with size of a single slide. On next and previous, we will change make the slides move within the container.
