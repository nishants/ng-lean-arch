Exercise 1. Extracting data from pages

This exercise introduces to

  • manipulating substring
  • accessing values from hash object
  • returning values from methods
  • mapping an array of values

In this exercise we need to read a page object to extract values from it. Following is an example of the page object 

page = {
    :name         => "Dam",
    :full_name    => "Dam Sam",
    :age          => 32,
    :address      => {:city => "Delhi", :from => {:day => 3, :month => 2, :year => 2014}},
    :past_addresses => [
        {:city => "Mumbai",    :from => {:day => 13, :month => 7, :year => 2012},   :to => {:day => 3, :month  => 2, :year => 2014}},
        {:city => "Chennai",   :from => {:day => 17, :month => 3, :year => 2010},   :to => {:day => 13, :month => 7, :year => 2012}},
        {:city => "Guwahati",  :from => {:day => 4,  :month => 2, :year => 2006},   :to => {:day => 17, :month => 3, :year => 2010}}
    ]
}
Hints
    • Get full name using page[:full_name] .
    • Get fist name using page[:name] .
    • You need to know how to find, replace strings, and how to remove leading and trailing spaces from a string.
    • Get age from page name using page[:age].
    • You can get the current year using Time.new.year.
    • You can get city name from page address.
    • You can get the past addresses using page[:past_addresses] .
    • Read about functions in ruby arrays. You might find map function useful.
    • From page[:past_addresses] and the address[:to] field, we already know when the adress was changed.
    • Use on of the array methods to chage the page[:past_addresses] to and array of dates in correct format.
